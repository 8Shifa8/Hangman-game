# Hangman Game

A classic Hangman word guessing game built in Python. Test your vocabulary skills by guessing letters to reveal a hidden word before running out of attempts!

## 🎮 How to Play

1. Run the game: `python hangman.py`
2. A random word will be hidden (displayed as underscores)
3. Guess one letter at a time by typing it and pressing Enter
4. If the letter is in the word, it will be revealed in all correct positions
5. If the letter is not in the word, you lose one attempt
6. You have **15 incorrect guesses** before the game ends
7. Type `exit` at any time to quit the game

## ✨ Features

- Random word selection from a list of 27 challenging words
- Tracks guessed letters to prevent repeats
- Displays current progress after each guess
- Shows remaining attempts
- Input validation (only single letters allowed)
- Exit option to quit mid-game

## 📋 Word List

The game includes interesting words like:
- python, unicorn, galaxy, falcon
- tyrannosaurus, metamorphosis
- lighthouse, rollercoaster
- narwhal, platypus, and more!
